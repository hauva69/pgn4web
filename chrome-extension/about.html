about the chess games viewer extension

The chess games viewer extension for Google Chrome is an interactive chessboard showing chess games from links to PGN URLs, from local PGN files and from PGN text, using context menus and page actions.

PGN (Portable Games Notation) files are commonly used by chess websites and by chess software to store and share chess games information.

context menu for links to PGN chess games URLs

A context menu shows chess games from links to PGN chess games URLs.

The mouse cursor icon highlights links to PGN chess games URLs when hovering on them.

try right-click this link to a PGN file and select the menu item for viewing the chess games.

live refresh mode

The live refresh mode reloads the games periodically until all games are ended, as designed for links to chess games live broadcasts.

A single large chessboard view is offered as well as a view with multiple small chessboards; click square H5 (assuming White on bottom) to switch between chessboard views.

During a live broadcast, the window title contains a set of three numbers, like "1.2.3 live broadcast" where the first number increases each time a new move is received for the visible games (acting as new move notification), the second is the overall number of live games and the third is the overall number of finished games.

try the PGN files used for demo are not from a current live broadcast; using the chess games viewer extension menu items and links for viewing games with live refresh always reports the live broadcast as already ended; instead this link opens a demo live broadcast using the single large chessboard view and this link opens a demo live broadcast using the multiple small chessboards view.

context menu for links to ZIP archives

A context menu for links to ZIP archives offers to inspect each ZIP archive for PGN chess games files.

Loading ZIP archives is a slow process; only the deflate and store compression methods are supported; encryption is not supported.

try right-click this link to a ZIP file and select the menu item for inspecting the ZIP file for viewing the chess games.

context menu for local PGN and ZIP files

A context menu allows showing chess games from local PGN files and local ZIP archives containing PGN files; drag and drop on the large chessboard view page also opens local files.

A browser bookmark can be created for easy access to the chess games viewer page for showing local PGN and ZIP files.

try first save locally the PGN file and the ZIP file at the links above, then right-click on the background of this page and select the menu item for opening local chess games files for viewing.

page action for links to PGN chess games URLs

A page action looks for links to PGN chess games URLs in the current page; if found, a black pawn page action icon appears in the address bar, clicking that icon a popup page allows for opening the links in the chess games viewer.

try the black pawn icon should have appeared at the right end of the address bar of this page; click that icon to display the popup page and try clicking the action icons.

user interface help

Using the keyboard or clicking squares interacts with the chessboards; for instance, click square F5 (assuming White on bottom) to display the chess puzzler of the day; hovering the mouse on each square displays an help summary. Click square H8 (assuming White on bottom) to display detailed help.

Errors alert messages are logged, such as failures to load PGN games, incorrect PGN games or incorrect FEN strings. When an error is encountered, the top left chessboard square will flash to signal the exception. The error alert log can be reviewed clicking the same top left chessboard square.

The chess games viewer extension does not have any configurable option, however each chessboard view remembers some settings between sessions.

single large chessboard view

Comment display within the game text: click square B7 (assuming White on bottom) to change whether comments are displayed or not into the move text; when comments are shown, click square C7 (assuming White on bottom) to change how comments are displayed respect to the move text, either inline or on separate lines.

Chessboard move highlight: click square D7 (assuming White on bottom) to toggle the highlighting of the last move's from/to squares.

multiple small chessboards view

Maximum number of shown chessboards: if more games are available than currently shown, the "m" command at the top right of the page allows increasing the number of shown chessboards; if more than one game is currently shown, the "l" command allows reducing the number of shown chessboards; select the number of chessboard that better fits the screen.

reset settings to default values

Click here to reset all settings to their default values. Reload any open chess games viewer page for the restored default settings to apply.

From the single large board view, click the chessboard top right square for user interface help, credits and license.

The chess games viewer extension for Google Chrome is part of the pgn4web project. The pgn4web project's wiki provides more information, including a support page for the chess games viewer extension with usage examples, answers to frequently asked questions and options for leaving feedback to the development team.
