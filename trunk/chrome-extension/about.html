about the chess games viewer extension

The chess games viewer extension for Google Chrome is an interactive chessboard showing chess games from links to *.PGN URLs, from local *.PGN files and from PGN text, using context menus and page actions.

  • A context menu shows chess games from links to *.PGN chess games URLs; PGN (Portable Games Notation) files are commonly used to store and share chess games information. The live refresh option reloads the games periodically until all games are ended; use for links to chess games live broadcasts; a single large chessboard view is offered as well as a view with multiple small chessboards. The mouse cursor icon highlights links to *.PGN chess games URLs when hovering on them.
  • A context menu allows showing chess games from local *.PGN files; drag and drop on the large chessboard view page also opens local *.PGN files.
  • A context menu shows chess games from PGN chess games text selecetd in the web page.
  • A page action looks for links to *.PGN chess games URLs in the current page; if found, a page action icon in the address bar allows for opening the links in the chess games viewer.

The top left chessboard square will flash in case of PGN data errors, click that square for further details.

Click on the viewer's chessboard top right square for user interface help, credits and license.

The chess games viewer extension for Google Chrome is part of the pgn4web project; visit the pgn4web project wiki for more information about the chess games viewer extension and for usage examples.
