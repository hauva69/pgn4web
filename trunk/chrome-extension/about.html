about the chess games viewer extension

features

The chess games viewer extension for Google Chrome is an interactive chessboard showing chess games from links to PGN URLs, from local PGN files and from PGN text, using context menus and page actions.

PGN (Portable Games Notation) files are commonly used by chess websites and by chess software to store and share chess games information.

context menu for links to PGN chess games URLs

A context menu shows chess games from links to PGN chess games URLs.

The mouse cursor icon highlights links to PGN chess games URLs when hovering on them.

live refresh option

The live refresh option reloads the games periodically until all games are ended, as designed for links to chess games live broadcasts.

A single large chessboard view is offered as well as a view with multiple small chessboards; click on square H5 (assuming White on bottom) to switch between chessboard views.

context menu for links to ZIP archives

A context menu for links to ZIP archives offers to inspect each ZIP archive for PGN chess games files.

Loading ZIP archives is a slow process; only the deflate and store compression methods are supported; encryption is not supported.

context menu for local PGN and ZIP files

A context menu allows showing chess games from local PGN files and local ZIP archives containing PGN files; drag and drop on the large chessboard view page also opens local files.

A browser bookmark can be created for easy access to the chess games viewer page for showing local PGN and ZIP files.

context menu for selected PGN chess games text

A context menu shows chess games from PGN chess games text selecetd in the web page.

page action for links to PGN chess games URLs

A page action looks for links to PGN chess games URLs in the current page; if found, a black pawn page action icon appears in the address bar, clicking that icon a popup allows for opening the links in the chess games viewer.

user interface help

Using the keyboard or clicking squares interacts with the chessboards; hovering the mouse on each square displays an help summary. Clicking square H8 (assuming White on bottom) displays detailed help.

Errors alert messages are logged, such as failures to load PGN games, incorrect PGN games or incorrect FEN strings. When an error is encountered, the top left chessboard square will flash to signal the exception. The error alert log can be reviewed clicking on the same top left chessboard square.

settings

The chess games viewer extension does not have any configurable option, however each chessboard view remembers some settings between sessions.

single large chessboard view

Comment display within the game text: click on square B7 (assuming White on bottom) to change whether comments are displayed or not into the move text; when comments are shown, click on square C7 (assuming White on bottom) to change how comments are displayed respect to the move text, either inline or on separate lines.

Chessboard move highlight: click on square D7 (assuming White on bottom) to toggle the highlighting of the last move's from/to squares.

multiple small chessboards view

Maximum number of shown chessboards: if more games are available than currently shown, the "m" command at the top right of the page allows increasing the number of shown chessboards; if more than one game is currently shown, the "l" command allows reducing the number of shown chessboards; select the number of chessboard that better fits the screen.

credits and license

From the single large board view, click on the chessboard top right square for user interface help, credits and license.

support

The chess games viewer extension for Google Chrome is part of the pgn4web project. The pgn4web project's wiki provides more information, including a support page for the chess games viewer extension with usage examples and options for leaving feedback to the development team.
